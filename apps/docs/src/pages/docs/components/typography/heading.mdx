# Heading

Headings are used for rendering headlines.
it renders an `<h2>` tag by default.

### Import

```js
import { Heading } from '@blitz-ui/react';
```

### Usage

<ComponentContainer>
  <Heading>I'm a Heading</Heading>
</ComponentContainer>

```jsx
<Heading>I'm a Heading</Heading>
```

### Changing font size

To increase the font size of the text, you can pass the `size` prop.
It apply a combination of fontSize and lineHeight for an optimal reading experience

<ComponentContainer>

### Override style

You can change the entire style of the text by className.

<ComponentContainer>
  <Heading className="text-primary-11">I'm a Heading</Heading>
</ComponentContainer>

```jsx
<Heading className="text-primary-11">I'm a Heading</Heading>
```

##### `size`

- Description: The size of the text. A combination of `font-size` and `line-height`.
- Type: `xs` | `sm` | `base` | `lg` | `xl` | `2xl` | `3xl` | `4xl` | `5xl` | `6xl` | `7xl` | `8xl` | `9xl`
- Default: `xl`
