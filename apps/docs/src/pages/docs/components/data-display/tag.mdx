# Tag

`Tag` component is used for items that need to be labeled, categorized,
or organized using keywords that describe them.

## Import

Blitz UI exports 5 Tag related components:

- **Tag**: The wrapper for all the tag elements.
- **TagLabel**: The label for tag's text content.
- **TagIconLeft**: The icon placed on the left side of the tag.
- **TagRightIcon**: The icon placed on the right side of the tag.
- **TagCloseButton**: The close button for the tag.

```js
import {
  Tag,
  TagLabel,
  TagIconLeft,
  TagRightIcon,
  TagCloseButton
} from '@blitz-ui/react';
```

## Usage

<ComponentContainer>
  <Tag>Sample Tag</Tag>
</ComponentContainer>

```jsx
<Tag>Sample Tag</Tag>
```

## Props

##### `variant`

- Description: The visual style of the tag.
- Type: `solid` | `subtle` | `outline`
- Default: `subtle`

##### `color`

- Description: The color of the tag.
- Type: `primary` | `info` | `success` | `warning` | `danger` | `neutral`
- Default: `neutral`

##### `size`

- Description: The size of the tag.
- Type: `sm` | `md` | `lg`
- Default: `md`

##### `shape`

- Description: The shape of the tag.
- Type: `rounded` | `square`
- Default: `rounded`
