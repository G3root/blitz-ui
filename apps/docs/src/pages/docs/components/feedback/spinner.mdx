# Spinner

Spinners provide a visual cue that an action is either processing, awaiting a
course of change or a result.

## Import

```js
import { Spinner } from '@blitz-ui/react';
```

## Usage

<ComponentContainer>
  <Spinner />
</ComponentContainer>

```jsx
<Spinner />
```

## Props

##### `size`

- Description: The size of the spinner.
- Type: `xs` | `sm` | `md` | `lg` | `xl`
- Default: `md`

##### `color`

- Description: The color of the spinner.
- Type: `primary` | `info` | `success` | `warning` | `neutral` | `danger` | `hiContrast` | `loContrast`
- Default: `hiContrast`
