# Icon Button

IconButton composes the `Button` component except that it renders only an icon

## Import

```js
import { IconButton } from '@blitz-ui/react';
```

## Usage

<ComponentContainer>
  <IconButton
    aria-label="Search database"
    icon={<SearchIcon className="h-5 w-5" />}
  />
</ComponentContainer>

```jsx
<IconButton
  aria-label="Search database"
  icon={<SearchIcon className="h-5 w-5" />}
/>
```

## Props

##### `aria-label` <Tag color="danger">Required</Tag>

- Description: A label that describes the button.
- Type: `string`

##### `icon`

- Description: The icon to be used in the button.
- Type: `JSX.Element`

##### `variant`

- Description: The visual style of the button.
- Type: `solid` | `flat` | `ghost` | `outline` | `link`
- Default: `solid`

##### `color`

- Description: The color of the button.
- Type: `primary` | `info` | `success` | `warning` | `danger` | `neutral`
- Default: `primary`

##### `shape`

- Description: The shape of the button.
- Type: `pill` | `square`
- Default: `square`

##### `size`

- Description: The size of the button.
- Type: `xs` | `sm` | `md` | `lg` | `xl`
- Default: `md`

##### `disabled`

- Description: If `true`, the button will be disabled.
- Type: `boolean`
- Default: `false`

##### `loading`

- Description: If `true`, the button will show a loader.
- Type: `boolean`
- Default: `false`

##### `loadingText`

- Description: The label to show in the button when `loading` is true.
- Type: `string`

##### `spinner`

- Description: Replace the spinner component when `loading` is `true`.
- Type: `JSX.Element`
