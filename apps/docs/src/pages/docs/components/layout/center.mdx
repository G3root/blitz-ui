# Center

Center is a layout component that centers its child within itself.

### Import

```js
import { Center } from '@blitz-ui/react';
```

### Usage

Put any child element inside it, give it any `width` or/and `height`, it'll
ensure the child is centered.

<ComponentContainer className="bg-danger-9">
  <Center className="h-12">This is the Center</Center>
</ComponentContainer>

```jsx
<Center className="h-12">This is the Center</Center>
```
